# gopals-python-scripts

Minimal collection of Python scripts for data validation used by the gopals project.

## Purpose

This repository stores small, reusable data validation scripts (CSV, JSON, DB queries) and helpers used in pipelines and local checks.

## Getting started

1. Create a virtual environment:
   python -m venv .venv
   source .venv/bin/activate

2. Install dependencies:
   pip install -r requirements.txt

## Repository layout

- README.md — this file
- requirements.txt — runtime dependencies (pandas only)
- scripts/ — folder for Python scripts
